# 🏝️ 점프 퀴즈 아일랜드

초등 1~3학년을 위한 3D 점프맵 학습 게임 (수학 · 국어 · 영어)

## 🎮 플레이 방법

GitHub Pages 주소로 접속하면 바로 플레이됩니다:
`https://<깃허브아이디>.github.io/<저장소이름>/`

- **PC**: 방향키 또는 WASD 이동 · 스페이스 점프
- **태블릿·폰**: 왼쪽 가상 조이스틱 이동 · 오른쪽 버튼 점프

## ✨ 게임 구성

- **스테이지 3개**: 🏝️ 하늘 섬 → 💎 반짝 동굴 → 🚀 둥둥 우주 (앞 스테이지를 깨야 다음이 열림)
- **퀴즈 문**: 스테이지마다 6개, 수학·국어·영어 순환. 정답이 적힌 문으로 들어가면 통과
- **뽑기 상점**: 모은 코인으로 모자·펫·셔츠 뽑기 (19종, 4개 등급)
- **아이별 프로필**: 1·2·3학년 버튼이 각자의 저장 슬롯 (지갑·아이템·진행도 분리)
- **부모님 리포트**: 시작 화면의 리포트 버튼을 1.5초 이상 길게 누르면 열림 — 아이별 정답률과 자주 틀린 문제 확인

저장은 기기(브라우저)별로 이루어집니다. 아이가 쓰는 태블릿마다 기록이 따로 유지돼요.

## 🔄 업데이트 방법

1. 새 `index.html` 파일을 받는다
2. 이 저장소에서 **Add file → Upload files**로 같은 이름으로 업로드 (기존 파일이 교체됨)
3. Commit 후 1~2분 뒤 자동으로 게임에 반영 (시작 화면 하단의 버전 표시로 확인)
